# T1 任务拆分

<!-- 作用：把“三资管理助手简明化、场景化、双端适配全面优化升级”拆成可并行开发的任务。 -->

## 总目标

实现适配 PC 和移动端的三资管理智能问答助手，支持简明回答、资料来源展示、推荐追问和本地知识库检索。

## 成员 A：项目管理 + 整体架构 + main 分支管理

- 初始化项目结构。
- 建立前后端基础工程。
- 定义 API 数据契约。
- 编写架构和 Git 协作规范。
- 审核并合并成员 PR。

## 成员 B：PC 端智能问答界面

- 完成桌面端问答布局。
- 实现输入框、回答区、来源区、追问区。
- 优化长回答阅读体验。

## 成员 C：移动端适配

- 完成移动端单列布局。
- 优化底部输入栏和触控区域。
- 检查小屏幕回答、来源和追问展示。

## 成员 D：知识图谱体系

- 整理政策法规、工作手册、业务规范、平台说明。
- 扩展 `backend/data/knowledge_base.json`。
- 后续可把检索升级为向量库或图数据库。
